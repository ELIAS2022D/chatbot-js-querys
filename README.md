# Bot de WhatsApp - (Respuestas Fijas)

Este bot responde automáticamente a mensajes de WhatsApp según palabras clave.

## 🧩 Requisitos

- Node.js 18 o superior
- Render para montar la aplicación y tener acceso a consola

## 🚀 Cómo usar

1. Instalá dependencias:

```
npm install whatsapp-web.js qrcode-terminal
```

2. Iniciá el bot:

```
npm start
```

3. Escaneá el QR desde WhatsApp (en tu celular: WhatsApp > Dispositivos vinculados).

4. ¡Listo! Probalo enviando por WhatsApp mensajes como:

- "Pagos"
- "Datos bancarios"
- "Hablar con un asesor"

## 🛠 Personalización

Editá `respuestas.json` para cambiar o agregar respuestas automáticas.
